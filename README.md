A simple and responsive calculator built using HTML, CSS, and Vanilla JavaScript.
This project performs basic arithmetic operations such as addition, subtraction, multiplication, and division with a clean and user-friendly interface.

✨ Features
Basic arithmetic operations (+, −, ×, ÷)
Responsive grid-based layout
Real-time display update
Clear (C) functionality
Error prevention for incomplete inputs
Beginner-friendly and easy-to-read code
🛠️ Technologies Used
HTML5 – Structure
CSS3 (Grid & Flexbox) – Styling and layout
JavaScript (Vanilla JS) – Logic and functionality
🎯 Purpose

This project was created to practice DOM manipulation, event handling, and JavaScript logic building without using any external libraries or frameworks.
